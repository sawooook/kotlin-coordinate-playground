# 기능 정리
1. 현재 보유하고 있는 차량은 Sonata 2대, Avante 1대, K5 2대로 총 5대의 차량이다.
2. 예약할때 고객으로부터 대략적인 이동거리를 입력 받는데 이 이동거리를 통해 연료를 주입함
3. 차량별로 주입해야할 연료량 보고서 생성 해야함

Sonata : 10km/리터
Avante : 15km/리터
K5 : 13Km/리터
